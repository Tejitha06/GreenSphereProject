🌿 GreenSphere Frontend Integration Guide

Complete guide to integrate your HTML files with the Flask backend for plant identification and disease detection.

✅ Quick Start

Expected Time: 5-10 minutes

  1. Copy api-client.js to your frontend directory
  2. Add script tags to identify.html and disease.html
  3. Replace the identifyPlant() and detectDisease() functions
  4. Start the Flask backend: python run.py
  5. Open your HTML files in browser

📝 Step 1: Update identify.html

Add these two script tags BEFORE the closing </body> tag:

<script src="api-client.js"></script> <script src="identify-integration.js"></script>

Then replace the entire identifyPlant() function with the code from identify-integration.js

⚠️ Important: Keep the rest of the HTML and CSS intact. Only replace the JavaScript identifyPlant() function and the plantData object is still used for common plants reference.

📝 Step 2: Update disease.html

Add these two script tags BEFORE the closing </body> tag:

<script src="api-client.js"></script> <script src="disease-integration.js"></script>

Then replace the detectDisease() function with the code from disease-integration.js

🚀 Step 3: Start the Backend

# Navigate to backend directory cd backend # Activate virtual environment (Windows) venv\Scripts\activate # Or on Mac/Linux source venv/bin/activate # Install dependencies (if not done yet) pip install -r requirements.txt # Set your PlantID API key in .env file # PLANTID_API_KEY=your_api_key_here # Run the server python run.py

🔧 Configuration

Get PlantID API Key

  1. Visit https://plant.id/
  2. Sign up for a free account
  3. Go to your dashboard and copy your API key

Configure .env

# Create file: backend/.env PLANTID_API_KEY=your_api_key_from_plant_id PLANTID_API_URL=https://api.plant.id/v3 FLASK_ENV=development FLASK_DEBUG=True CORS_ALLOWED_ORIGINS=http://localhost:5000,http://localhost:3000,http://127.0.0.1:5000

🐛 Troubleshooting

Error: "PlantID API key is not configured"

  • Make sure you created backend/.env file
  • Add your API key: PLANTID_API_KEY=your_key_here
  • Restart the Flask server

Error: CORS error in browser console

  • Check that your API URL is correct in api-client.js
  • Verify backend is running on http://localhost:5000
  • Check CORS_ALLOWED_ORIGINS in .env includes your frontend URL

Error: "NetworkError" or timeout

  • Make sure Flask backend is running
  • Check internet connection (PlantID API needs external connection)
  • Try with a smaller image file (< 5MB)

Error: 500 Internal Server Error

  • Check the error message in Flask terminal
  • Verify image file is valid and supported format
  • Check PlantID API key is correct

📱 Browser Console Debugging

If something doesn't work, open browser console (F12) and check for errors:

  • Network tab: Check API request/response
  • Console: Look for JavaScript errors
  • Use greensphereAPI.getHealth() to test API connection

🎨 Customization

Change API URL

If running backend on different port/host, edit api-client.js:

const greensphereAPI = new GreenSphereAPI('http://your-backend-url:5000/api');
